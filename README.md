# أوركيدة — موقع لحجز المقاعد والدورات التدريبية

**أوركيدة** هو موقع إلكتروني يتيح للمستخدمين حجز مقاعدهم في مساحات العمل المختلفة والتسجيل في دورات تدريبية متنوعة بطريقة سهلة وسريعة. تم تصميم الموقع ليكون واجهته واضحة وسلسة، مع إمكانية متابعة الحجزات والتسجيلات مباشرة.

## المميزات

- **حجز المقاعد:** يمكن للمستخدمين اختيار مقاعد متاحة وحجزها مباشرة.  
- **التسجيل في الدورات:** يوفر الموقع مجموعة من الدورات التدريبية مع معلومات عن المدة وعدد المقاعد المتاحة.  
- **واجهة سهلة الاستخدام:** تصميم متجاوب وواجهة تفاعلية باستخدام HTML وCSS وJavaScript.  
- **التخزين السحابي:** جميع بيانات الحجز والتسجيل تُخزن في **Firebase Realtime Database** لضمان الحفاظ على المعلومات وتحديثها مباشرة.  
- **تأثيرات ديناميكية:** القوائم المنسدلة والمودالات تعمل بسلاسة مع تأثيرات بصرية لطيفة.  

## التقنيات المستخدمة

- **Frontend:** HTML, CSS, JavaScript  
- **Backend/Database:** Firebase Realtime Database  
- **Hosting:** يمكن رفع الموقع على GitHub Pages أو أي استضافة أخرى  

## كيفية الاستخدام

1. افتح الصفحة الرئيسية `index.html`.  
2. استخدم زر "احجز مقعد الآن" للوصول إلى صفحة المقاعد وحجز أي مقعد متاح.  
3. استخدم زر "انضم لدورة تدريبية" للتسجيل في الدورات المتاحة.  
4. يمكن متابعة الحجزات والتسجيلات مباشرة في Firebase Realtime Database.  

## ملاحظات

- جميع البيانات المخزنة مؤمنة ومباشرة في Firebase، ويمكن عرضها في أي وقت عبر لوحة Firebase.  
- التصميم متجاوب ويمكن استخدام الموقع على الهواتف والكمبيوتر.
